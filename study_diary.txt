2016/02/22　Githubの基本の勉強をした

2016/02/23　Atomを導入、tigをインストール
pull requestのプラクティスまで進んだので、
再度手順に沿ってトライしてみたけどやはり同じエラー／(^o^)＼
@poyopoyoさんが日報にコメントしてくれたbrew install q4
を試してみるも、解決されず…奮闘するもダメだった(´；ω；｀)

2016/02/24 pull requesutがどうも上手く行かないので、
帰ってからクリーンインストールして再度チャレンジする事にした。
今日は一日MySQLについて学んだ。以前のプラクティスにも出てきた
から少し触った事があったけど、複数のテーブルのデータを関連付け？
して表示するのは初めて使ってみたと思う。楽しみながら学習出来て
集中力も割りと続いたので良かった。

2016/02/25　アップデートした事で、ようやく環境構築完成！！！
無事出来て本当に良かった…今日はHTTPの勉強を始めていたので、
プルリクは明日行う事に。HTTPを勉強して普段何気なくURLを
入力してサイトにアクセスしているけど、裏ではリクエストを
送ってレスポンスが返される事でサイトを閲覧する事が出来る
んだなぁ、と学んだ。裏では色々なものが動いて便利に使う事
ができているものって沢山あるよなぁ、と思った(ヽ´ω`)

2016/02/26　初めてリモートリポジトリにpull requestした。
ローカルリポジトリを触る分には慣れてきたけど、リモートリポジトリ
だとコンフリクトが起きたりと複数人で共用しているからこその問題が
発生するので、コンフリクトが起きても焦らず解消出来るようになりたい。
MySQLの日本語が？？？になってしまうのでutf8(日本語設定)にした
のに、insertしてselectで確認してみるもやはり？？？のまま。
どうすれば上手く日本語が表示されるようになるんだろうか…
まだまだ知らない事だらけで経験と勉強が必要だな。
出来ないとイライラするし、凄い辛い思いをするけどがんばろう。

2016/02/29　26日にやったMySQLの日本語設定は上手く出来ていた。
…では何故日本語が表示されなかったかと言うと、今まで操作
していたデータベースには設定が反映されないらしい。なので
新しくデータベース・テーブルを作りinsertしたところ、
上手く日本語が反映されていた！良かった良かった(´；ω；｀)
TDDが何やら難しそう。何でわざわざ成功しないコードを書く
必要があるのかがよく理解出来なかった。色々参考になる記事
を読んで学習を積むしかないな。

2016/03/01　今日から3月…あっという間過ぎて怖い。
Gitについてもっと復習が必要だと思った。リモート
リポジトリは自由に使えないので、コンフリクトとか
起こる事も多々あるから、そういう時に慌てないように
する必要があると思った。Railsも少し勉強してみたけど、
以前勉強した事を若干思い出してきた。だけどまだまだ…

2016/03/04　ここ最近GitHubに勉強の記録をかけていなかった。
納会で「使えば慣れる」という話を聞いたので、やっぱり何事も
経験を積むしかないなぁ、と思った。自分一人で使うのと、会社
で実践するのは違うと思うけど、一人ででも使っておいた方が
GitHub自体には慣れる事が出来ると思うので、また今日から勉強
の記録を書いていこうと思う。

2016/03/14　久しぶりにGitを使った。約10日ぶり！
Gitよりご無沙汰していたCSSを触ったけど、デザインを
当てていくのはやっぱり難しい…試行錯誤して上手い事
綺麗なデザインにしていくのは楽しい半面大変だった。

2016/03/15　Atomのプラグインは色々便利なものが多いので、
何か入れておくと良さそうなものを探していたら、ドット
インストールがヒットした。ドットインストールはAtomの動画
まで用意されているのか…と驚かされた。凄いぞ、ドットインストール。
command + k 矢印で指定した方向にタブを分割する事が出来る、
command + option + ←もしくは→でタブ移動する事が出来るという事を
知った。プラグインではないけど。

2016/03/16　今日は朝異常に眠かったので、ギリギリまで寝て
しまった。昨日はいつもより早く寝たのに不思議！(=_=)
ドットインストールでAtomの動画を見て、同じワードを一気に
置換する方法、フォルダ全体から検索、snippetsを知った。
何事も使わないと慣れないから、使って身体に身につける
しかない。似たようなコマンドが多いから大変だけど…
早く就職したいところだけど、まだまだ全然何も出来ないから
時間がかかりそうだなぁ。それが心配…普通に1年以上かかりそうだ。
途中で心が折れないようにしないと。変に焦ってしまう(；´∀｀)
